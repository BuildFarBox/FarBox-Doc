## FarBox
FarBox是一个很棒的博客平台。

但当你看到doc.farbox.com的时候，则会发现，远不止如此。

此时的FarBox已经脱离了 [WordPress](https://wordpress.org/)、[Ghost](https://ghost.org/)、[Jekyll](http://jekyllrb.com/) 所属的各自领域。

应该有那么一天，他们会抱着复杂的心情说: 「难以置信」！
